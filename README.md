bundle
======

Web-based filemanager.

Files are stored below the root of the web server, therefore the files are not accessible through the URL.
Each user gets a directory under /var/webapps/storage/bundle/

##Possible actions

- Add files
- Add directory
- Go to sub-directory
- Preview textfiles
- Preview images
- Download files

##Dependencies
